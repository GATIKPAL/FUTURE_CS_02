
## Objective
Analyze phishing emails, identify risk indicators, and provide prevention guidelines for end users.

## Tools Used
- Email Header Analyzer: https://toolbox.googleapps.com/apps/messageheader/
- MXToolbox Email Headers: https://mxtoolbox.com/EmailHeaders.aspx
- Google Docs / MS Word for reporting
- Virus Total: Url Analysis

## Analysis Approach
1. Collected phishing email samples.
2. Analyzed email headers for spoofing and domain mismatch.
3. Inspected links and domains for suspicious patterns.
4. Identified phishing indicators and classified risk.
5. Explained the attack in simple language.
6. Prepared prevention guidelines.
7. Compiled everything into a professional report.

## Repository Contents
- `report/` → Word and PDF report
- `samples/` → Sample phishing email text files
- `screenshots/` → Screenshots from analysis
